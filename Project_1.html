LESSON-1: The Basics of the Web and HTML

How the Web Works

The Web is the collection of text pages, digital photographs, music files, videos, animations etc. all of which you can access over the internet. A collection of web pages on the same computer is called a website. Every web page has highlighted phrases called "links" all over it.

HTML

HTML stands for Hyper Text Markup Language and is the standard markup language used to create the webpages. It is written in form of HTML elements consisting of tags enclosed in angle brackets.

Tags and Elements

HTML elements are written with a start tag and an end tag with the content in between.

Why Computers are Stupid

Computers are stupid because they understand only specific instructions written in specific format. This can create problems for the person performing simple funtions.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.

HTML tree-like structure

The "tree-like structure" comes from the fact that HTML elements can have other elements inside of them.

Indentations and Boxes

The placement of text farther to the right to separate it from surrounding text.All HTML elements can be considered as boxes. In CSS, the term "box model" is used when talking about design and layout.

Text Editors

A text editor is a type of program used for editing plain text files.Text editors are often provided with operating systems and software development packages, and can be used to change configuration files, documentation files and programming language source code.

LESSON-3: Adding CSS Style to HTML Structure

Avoiding Repitition

Avoiding repetition is important for a variety of reasons.By avoiding repitition it is easier to change style of things.

CSS

CSS stands for Cascading Style Sheets and they give programmers a way to control the style of related HTML elements. This is done by giving similar HTML elements the same class name and then specifying the style that should apply to that class.
